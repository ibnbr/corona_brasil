## Estrutura do arquivo

| Campo | Descrição |
| --- | --- |
| uid   | Id do Estado |
| estado | Nome do Estado |
| suspects | Casos Suspeitos |
| refuses | Casos Descartados |
| cases | Casos Confirmados |
| deads | Quantidade de Mortes |
| broadcast | Transmissão local confirmada |
